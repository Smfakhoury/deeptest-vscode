# DeepTest for VS Code

A VS Code extension that brings [DeepTest](https://github.com/microsoft/DeepTest) provenance data into your editor — specifications, bugs, hazards, tests, and behavioral claim negotiation.

## Features

- **📋 Specifications overlay** — gutter icons and CodeLens showing pre/postconditions per function
- **🐛 Bug diagnostics** — inline squiggles with severity from DeepTest triage
- **⚠️ Hazard decorations** — warnings for overflow, null deref, bounds issues
- **🔍 Provenance panel** — cursor-following panel showing specs, bugs, tests, hazards for the current function
- **💬 @deeptest Chat participant** — Copilot Chat integration for spec review and behavioral claim negotiation

## Chat Commands

| Command | Description |
|---------|-------------|
| `@deeptest /specs <function>` | Show specifications for a function |
| `@deeptest /bugs` | List all bugs grouped by severity |
| `@deeptest /review` | Start behavioral claim negotiation for current function |
| `@deeptest <question>` | Free-form Q&A grounded in provenance data |

## Prerequisites

- VS Code ≥ 1.100.0
- [GitHub Copilot Chat](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat) extension
- A `.deeptest/report/provenance.json` file in your workspace (generated by `deeptest run`)

## Input

The extension consumes `.deeptest/report/provenance.json`, produced by the DeepTest [report skill](https://github.com/microsoft/DeepTest/tree/main/.github/skills/report). Schema: [`provenance.schema.json`](https://github.com/microsoft/DeepTest/blob/main/.github/skills/report/schema/provenance.schema.json).

## Development
